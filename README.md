# Калькулятор токарных фасок (PWA)

Простое веб-приложение для быстрого расчёта параметров токарной фаски прямо с телефона у стойки Fanuc (или любой другой).

## Возможности

- Расчёт по формулам прямоугольного треугольника: катет по **Z** (осевой), катет по **X** (радиальный), гипотенуза (длина фаски).
- Угол можно указать **от осевого катета** (типовое обозначение «2×45°» — между торцом и фаской) или **от радиального катета**.
- Наружная фаска (вал) и внутренняя фаска (в отверстии) — переключатель меняет иллюстрацию.
- Интерактивная **SVG-схема** контура детали с показом Z, X, угла и линии фаски.
- Если указать исходный диаметр D, приложение покажет итоговый диаметр D2 после фаски (для наружной — D1 − 2X, для внутренней — D1 + 2X).
- **PWA**: ставится на домашний экран iPhone, работает оффлайн после первой загрузки.

## Быстрый запуск локально

Достаточно любого статического сервера (service worker не работает с `file://`).

Через Python:

```powershell
cd chamfer-calc
python -m http.server 8080
```

Или через Node (`npx serve`):

```powershell
cd chamfer-calc
npx serve -l 8080 .
```

Открыть в браузере: http://localhost:8080

Если телефон и компьютер в одной Wi‑Fi сети — можно открыть на iPhone по IP компьютера, например `http://192.168.1.10:8080`.

## Публикация на GitHub Pages (бесплатный доступ с любого места)

Это самый удобный вариант — ссылка будет доступна с iPhone из любой сети.

1. Создайте новый репозиторий на GitHub, например `chamfer-calc`.
2. Положите в него всё содержимое папки `chamfer-calc/` (файлы `index.html`, `app.js`, `styles.css`, `manifest.webmanifest`, `sw.js`, иконки).
3. Запушьте в ветку `main`.
4. В GitHub → **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main`, папка `/ (root)` → Save.
5. Через ~1 минуту сайт будет доступен по адресу `https://<ваш-ник>.github.io/chamfer-calc/`.

### Команды (пример)

```powershell
cd chamfer-calc
git init
git add .
git commit -m "init chamfer calculator"
git branch -M main
git remote add origin https://github.com/<ваш-ник>/chamfer-calc.git
git push -u origin main
```

## Установка на iPhone как приложение

1. Откройте сайт в **Safari** (именно Safari — Chrome на iOS не умеет ставить PWA).
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх).
3. Выберите «На экран «Домой»» (Add to Home Screen).
4. Задайте имя (по умолчанию «Фаски») и нажмите «Добавить».

Теперь на главном экране появится иконка; при запуске приложение открывается в полноэкранном режиме без адресной строки и **работает оффлайн**.

## Формулы (справочно)

Обозначим β — угол между гипотенузой фаски и осевым катетом (осью Z).

- Если на чертеже угол отсчитывается от осевого катета: `β = α`.
- Если от радиального: `β = 90° − α`.

Тогда:

- `X = Z · tan(β)`
- `Z = X / tan(β)`
- `гипотенуза = Z / cos(β) = X / sin(β)`
- Изменение диаметра: `ΔD = 2·X` (X — приращение по **радиусу**).
- Наружная фаска: `D2 = D1 − 2·X`. Внутренняя фаска: `D2 = D1 + 2·X`.
